# M2-Quest-Poly

Mission on marble polymorphism.  
Just approach the NPC, choose a mob and kill the required number of pieces. Marble will automatically go to EQ  

Setting
```
[1] = {101, 100},
[2] = {502, 100},
[3] = {2001, 100},
[4] = {636, 150}
```

```
[id mission, 1,2,3 etc...] = {mob_id, count_kill},
```
